pyfos_brocade_access_gateway - PyFOS module to provide REST support for an Access Gateway.¶

The pyfos_brocade_access_gateway module provides REST support for an Access Gateway.

class pyfos_brocade_access_gateway.device_list¶

This class provides information about the devices logged in to the Access Gateway.

Important Class Members:

Attribute Name Description Frequently Used Methods
wwn The port WWN of the device. peek_wwn()
fcid The Fibre Channel ID of the device. peek_fcid()
f-port The F_Port to which the device is connected. peek_f_port()
n-port The N_Port to which the device F_Port is mapped. peek_n_port()

Object Methods

get()¶

Returns a device_list object with values for all attributes. The object can be printed using pyfos_utils.response_print().

Parameters:session – The session handler returned by pyfos_auth.login().

Example Usage of the Method to Get Information for All WWNs:

Object Methods

get(session)¶

Returns an n_port_map object or a list of objects filled with attributes gathered from the Access Gateway. If the optional N_Port is given, either an object matching the name of the port is returned or an empty object is returned if no match is found.

Each object can be printed using pyfos_util.response_print(), and individual attributes can be accessed through peek methods.

Parameters:session – The session handler returned by pyfos_auth.login().
Return type:An n_port_map object if there is only one N_Port configured in the switch or a list of objects if there are more than one.
patch(session)¶

Applies configurable attribute(s) within the object to an N_Port map of the Access Gateway. This method configures a specified list of F_Ports to be mapped (static/configured) to a given N_Port. This new map configuration will overwrite any existing map configuration for the specified N_Port. Also, this method is used to update the failover and failback configuration for a given N_Port.

The following is an example using individual sets:

# initialize the Access Gateway object
nport = pyfos_brocade_access_gateway.n_port_map();
# set the failover-enabled attribute to
# enable failover for N_Port 0/47 on the Access Gateway
nport.set_n_port("0/47")
nport.set_failover_enabled("1")
# execute the HTTP patch method to apply the object to the
# switch connected through the session
nport.patch(session)
Parameters:session – The session handler returned by pyfos_auth.login().
Return type:A dictionary of errors or success information.
post(session)¶

Adds members to a list attribute of the object. This method is used to add a specified list of F_Ports (configured and static) to a given N_Port.

Example Usage of the Method to Create New Mappings:

# initialize the Access Gateway object
nport = pyfos_brocade_access_gateway.n_port_map();
# set the failover-enabled attribute to
# enable failover for N_Port 0/47 on the Access Gateway
nport.set_n_port("0/47")
new_nport.set_configured_f_port_list_f_port("0/1;0/2");
result = new_nport.post(session)
Parameters:session – The session handler returned by pyfos_auth.login().
Return type:A dictionary of errors or a success response.
delete(session)¶

Deletes members from a list attribute of the object. This method is used to remove or clear the specified list of configured and static F_Ports from the mapping for N_Port.

Example Usage of the Method to Create New Mappings:

# initialize the Access Gateway object
nport = pyfos_brocade_access_gateway.n_port_map();
# set the failover-enabled attribute to
# enable failover for N_Port 0/47 on the switch
nport.set_n_port("0/47")
new_nport.set_configured_f_port_list_f_port("0/1;0/2");
result = new_nport.delete(session)
Parameters:session – The session handler returned by pyfos_auth.login().
Return type:A dictionary of errors or a success response.

post()¶

Creates an entry or adds members. The required fields are set within the object using the attribute’s set method. This method is used to create a new port group with a group of N_Ports or to add N_Ports or F_Ports to an existing port group.

Example Usage of the Method to Create a New Port Group:

portgroup_obj =
    pyfos_brocade_access_gateway.port_group()
portgroup_obj.set_port_group_id(1)
portgroup_obj.set_port_group_n_ports_n_port("0/40;0/41")
portgroup_obj.post(session)

The above example will add N_Ports if the port group with ID 1 exists already.

Parameters:session – The session handler returned by pyfos_auth.login().
Return type:A dictionary of errors or a success response.
patch()¶

Replaces entry members. The required fields are set within the object using the attribute’s set method. This command is used to replace the existing member N_Ports or F_Ports, the rename port group, and the enable or disable port group modes.

Example Usage of the Method to Enable Load Balancing Mode in a Port Group:

portgroup_obj =
    pyfos_brocade_access_gateway.port_group()
portgroup_obj.set_port_group_id(1)

Parameters:session – The session handler returned by pyfos_auth.login().
