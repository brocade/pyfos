pyfos_brocade_time - PyFOS module to provide rest support for time server.¶

The pyfos_brocade_time provides a REST support for Time Server.

class pyfos_brocade_time.clock_server(dictvalues={})¶
This class provides NTP clock server information and also can configure the
list of NTP clock server or LOCL.

Important class members:

Attribute name Description Frequently used methods
ntp-server-address/server-address NTP server address list or LOCL set_ntp_server_address_server_address() peek_ntp_server_address_server_address()
active-server Active server address or LOCL peek_active_server()

Object methods

static get(session)¶
Return a clock_server object filled with TS
Clock Server attributes.

Each object can be printed using pyfos_util.response_print() and individual attributes accessed through peek methods.

Parameters:session – session handler returned by pyfos_auth.login()
Return type:clock_server object. Dictionary in case of error.
patch()¶

Replace existing configuration. Fields involved are set within the object using attribute’s set method. This command is used to replace the existing NTP clock server configuration.

Example usage of the method to configure ntp clock server: .. code-block:: python

ts_obj = pyfos_brocade_time.clock_server() tz_obj.set_ntp_server_address_server-address(“1.2.3.4”,”5.6.7.8”) tz_obj.patch(session)
Parameters:session – session handler returned by pyfos_auth.login()
Return type:dictionary in case of error or success response

class pyfos_brocade_time.time_zone(dictvalues={})¶
This class provides system time zone information and also can configure the
time zone by both name and offset values.

Important class members:

gmt-offset-minutes Minutes offset values set_gmt_offset_minutes() peek_gmt_offset_minutes()

patch()¶

Replace existing configuration. Fields involved are set within the object using attribute’s set method. This command is used to replace the existing time zone configuration.
