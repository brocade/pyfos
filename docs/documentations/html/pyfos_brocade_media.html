Object Methods

get(session, name=None)¶

Returns a media_rdp object or a list of objects with the attributes gathered from port media. If an optional name is given, returns either an object matching the name of the port or an empty object with the corresponding error if no match is found.

Each object can be printed using pyfos_util.response_print(), and individual attributes can be accessed through peek methods.

Parameters:
  • session – The session handler returned by pyfos_auth.login().
  • name – The interface slot port details in the format of interface/slot/port.
Return type:

A media_rdp object if the name is provided or a list of objects if there is more than one.
