class pyfos_brocade_extension_ip_route.extension_ip_route(dictvalues=None)¶

Class of extension_ip_route

Description extension_ip_route

Represents static IP route on the IP interface defined on extension blade or system.

Important class members of extension_ip_route:

Attribute Name Description Frequently Used Methods
dp-id Extension Data Path Processor ID associated with the IP Route. Based on platform either it will have a single DP or dual DP. In case of single DP only DP0 is supported, and in case of dual DP both DP0 and DP1 are supported. 0 : DP0 1 : DP1 peek_dp_id() set_dp_id()
ip-prefix-length The prefix length operator for the destination IP address. Once set, prefix length can not be changed. peek_ip_prefix_length() set_ip_prefix_length()
name The name of the interface. peek_name() set_name()
ip-address Specifies the destination IPv4/IPv6 address. peek_ip_address() set_ip_address()
ip-gateway Specifies the IP address of an IP router that can route packets to the destination IP address. The gateway address must be on the same IP subnet as one of the port IP addresses. This operand is optional with IPv6 addresses. Once set, IP gateway can not changed. peek_ip_gateway() set_ip_gateway()
status-flags Iproute Flags: U = Usable G = Gateway H = Host C = Created (Interface) S = Static L = LinkLayer peek_status_flags()
