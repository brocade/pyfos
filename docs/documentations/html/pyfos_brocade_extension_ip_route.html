pyfos_brocade_extension_ip_route - PyFOS module for extension IP routes.¶

The pyfos_brocade_extension_iproute module provides REST support for IP route extension objects.

class pyfos_brocade_extension_ip_route.extension_ip_route(dictvalues={})¶

Class of extension_iproute

Important Class Members:

Attribute Name Description Frequently Used Functions
name The slot/port name of the GE_Port. peek_name() set_name()
ip-address The IPv4/IPv6 address. peek_ip_address()
dp-id The data-path processor ID. peek_dp_id() set_dp_id()
ip-prefix-length The IP prefix length. peek_ip_prefix_length() set_ip_prefix_length()
ip-gateway The IP address of the gateway. peek_ip_gateway()
status-flags The IP interface flags. peek_status_flags()

Object Functions

get()¶

Fills the object with values for all the attributes. Once filled, the object can be printed using pyfos_utils.response_print().

Parameters:session – The session handler returned by pyfos_auth.login().
Return type:A dictionary of errors or a success response.
