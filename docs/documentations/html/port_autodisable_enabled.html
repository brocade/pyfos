port_autodisable_enable - PyFOS util for specific port op use case.¶

The port_autodisable_ebable provides for specific port op use case.

This module is a stand-alone script that can be used to set/reset a port as a port.

  • Infrastructure options:
    • -L=<login>: Login ID. If not provided, an interactive
      prompt will request one.
    • -P=<password>: Password. If not provided, an interactive
      prompt will request one.
    • -i=<IP address>: IP address.
    • -u=<user name>: The string name to be assigned to switch.
    • -f=<VFID>: VFID or -1 if VF is disabled. If unspecified,
      a VFID of 128 is assumed.
  • Util scripts options:
    -n,–name=NAME Port in slot/port. -a,–port_autodisable_enabled=AUTODISABLE_ENABLED Set “port-autodisable-enabled” <0|1>.
  • Outputs:
    • Python dictionary content with RESTCONF response data.
