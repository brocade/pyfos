Object Methods

get()¶

Returns trunk objects for all trunk groups gathered from the switch. When optional params group and source port params are passed, returns either an object matching the group and source port or if no match is found, returns an empty object. When these optional params are passed, both should be set. The object can be printed using pyfos_utils.response_print().

Parameters:session – The session handler returned by pyfos_auth.login().
Return type:A dictionary of errors or a trunk object.

Object Methods

get()¶

Parameters:session – The session handler returned by pyfos_auth.login().
Return type:A dictionary of errors or trunk_area object.
post()¶

Create an entry. The required fields are set within the object using the attribute’s set method. This command is used to create a new F_Port trunk group or add ports to an existing trunk group.

Parameters:session – The session handler returned by pyfos_auth.login().
Return type:A dictionary of errors or a success response.
delete()¶

Delete an entry or entry members. The required fields are set within the object using the attribute’s set method. This command is used to delete a trunk group or delete the existing members of the trunk group. If all trunk ports are specified, the whole trunk group is deleted.

Parameters:session – The session handler returned by pyfos_auth.login().
Return type:A dictionary of errors or a success response.
