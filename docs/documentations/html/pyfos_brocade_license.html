pyfos_brocade_license - PyFOS module to provide rest support forLicenses installed on the specific switch in the fabric.¶

The pyfos_brocade_license provides the REST support for License.

class pyfos_brocade_license.license(dictvalues={})¶

Class of licenses installed on a switch

Important class members:

Attribute name Description Frequently used methods
name The license key for one or morefeatures installed on the switch peek_name()
features/feature The list of features that that are integrated in a single license key peek_features_feature()
capacity The capacity of the license installed on the switch.This field is displayed only for the capacity based license in the output. peek_capacity()
consumed The number of consumed slots of the license installed on the switch. peek_consumed()
configured-blade-slots/ configured-blade-slot The list of Configured Blade Slots of the specific license installed on the switch. peek_configured_blade_slots_configured_blade_slot()
expiration-date The expiry date of the specific license installed on the system. the format of the date is ‘MM/DD/YYYY’. peek_expiration_date()

Object methods

classmethod get(session)¶

Returns a license object filled with attributes gathered throuth the session passed in. Each object can be printed using pyfos_util.response_print() and individual attributes accessed through peek methods.

Parameters:session – session handler returned by utils.brcd_util.getsession()
Return type:a license object
