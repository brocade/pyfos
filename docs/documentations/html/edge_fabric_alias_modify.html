edge_fabric_alias_modify - PyFOS util to modify edge fabric alias¶

The edge_fabric_alias_modify PyFOS util to modify edge fabric alias

Displays edge fabric id’s corresponding alias name

edge_fabric_alias_modify : usage

  • Infrastructure Options:
    • -i,–ipaddr=IPADDR: The IP address of the FOS switch.
    • -L,–login=LOGIN: The login name.
    • -P,–password=PASSWORD: The password.
    • -f,–vfid=VFID: The VFID to which the request is directed.
    • -s,–secured=MODE: The HTTPS mode “self” or “CA” [Optional].
    • -v,–verbose: Verbose mode [Optional].
  • Util Script Options:
    • –edge-fabric-id=EDGE-FABRIC-ID The ID of the fabric in which the LSAN zone was created
    • –alias-name=ALIAS-NAME Alias name of the specific edge fabric id.
  • Output:
    • Python dictionary content with RESTCONF response data.
edge_fabric_alias_modify.modify_edge_fabric_alias(session, edge_fabric_id, alias_name)¶

Modify edge_fabric_alias

Example Usage of the Method:

ret = edge_fabric_alias_modify.modify_edge_fabric_alias(session,            edge_fabric_id, alias_name)
print (ret)

Details:

edge_fabric_aliasObj = edge_fabric_alias()
