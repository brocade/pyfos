pyfos_brocade_operation_supportlink - PyFOS module for the supportlink operation.¶

The pyfos_brocade_operation_supportlink module provides REST support for
the supportlink operation.
class pyfos_brocade_operation_supportlink.supportlink(dictvalues={})¶

Class of the supportlink Operation

Important Class Members:

Attribute Name Description Frequently Used Methods
action Type of action - default/collect/send. set_action()

Object Methods

post()¶

Configures supportlink. The required fields are set within the object using the attribute’s set method. This method is used to trigger the supportlink operation in the switch.

Example Usage of the Method to Post the Supportlink Operation:

ss_obj =
    pyfos_brocade_operation_supportlink.supportlink()
ss_obj.set_action("default")
ss_obj.post(session)

The above example triggers the supportlink operation.

Parameters:session – The session handler returned by pyfos_auth.login().
Return type:A dictionary of errors or a success response.

Parameters:action – The action type of the remote server.
