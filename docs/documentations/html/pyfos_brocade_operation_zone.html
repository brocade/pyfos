pyfos_brocade_operation_zone - PyFOS module used for zoning RPC operations performed on a Brocade switch.¶

The pyfos_brocade_operation_zone module provides REST support for zoning RPC operations performed on a Brocade switch.

class pyfos_brocade_operation_zone.zone_operation_parameters(dictvalues=None)¶

Class of zone_operation_parameters

Important class members:

Attribute Name Description Frequently Used Methods
action The type of RPC zone action operation to perform. set_action() peek_action()
zone-object The zone object upon which the RPC zone action will be performed. set_zone_object() peek_zone_object()

Object Methods

post()¶

Configures the specified zone operation RPC. The required fields are set within the object using the attribute’s set method. This method is used to trigger the zone operation in the switch.

Parameters:session – The session handler returned by pyfos_auth.login().
Return type:A dictionary of errors or a success response.
