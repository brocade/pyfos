Welcome to PyFOS’ Documentation!¶

PyFOS contains open source Python modules to facilitate mangement of FOS switches.

PyFOS Infrastructure Modules

  • pyfos_auth - PyFOS module to facilitate establishing a session to a FOS switch.
  • pyfos_rest_util - PyFOS module to provide REST support for FOS objects.
  • pyfos_util - PyFOS module to provide utility functions.

PyFOS Manager Modules

  • pyfos_config_manager - PyFOS module to provide config related handling Support.
  • pyfos_class_manager - PyFOS module to Manage all pyfos classes.
  • pyfos_rule_manager - PyFOS module to provide rule based handling Support.
