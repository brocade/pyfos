Welcome to PyFOS’ Documentation!¶

PyFOS contains open source Python modules to facilitate mangement of FOS switches.

PyFOS Infrastructure Modules

  • pyfos_auth - PyFOS module to facilitate establishing a session to a FOS switch.
  • pyfos_rest_util - PyFOS module to provide REST support for FOS objects.
  • pyfos_util - PyFOS module to provide utility functions.
  • pyfos_version - PyFOS module to provide versioning functionality for rest classes and attributes.
  • pyfos_type - PyFOS module for pyfos_rest_util.rest_attribute to associate correct data type and yang type informations in pyfos along with support for correct data type conversion and validations support for peek_ and set_ methods.
  • pyfos_auth_token - PyFOS module to facilitate Auth-Token based session to a FOS switch.
