syslog_modify - PyFOS util to modify the syslog configuration on a switch.¶

The syslog_modify util provides option to modify the syslog configuration parameters on a switch.

This module is a stand-alone script that can be used to modify the syslog configuration on a switch.

  • Input:
Infrastructure Options:
-i,–ipaddr=IPADDR The IP address of the FOS switch.
-L,–login=LOGIN The login name.
-P,–password=PASSWORD The password.
-f,–vfid=VFID The VFID to which the request is directed [OPTIONAL].
-s,–secured=MODE The HTTPS mode “self” or “CA” [OPTIONAL].
-v,–verbose Verbose mode [OPTIONAL].
Util Script Options:
–server=The IP address of the server.
–port=The port number.
–secure-mode=True or false.
  • Output:
    • Python dictionary content with RESTCONF response data.
modify_syslog_server(session, facility):

Example Usage of the Method:

ret = syslog_modify.modify_syslog_server(session, server, port,
                                         secure_mode)
print (ret)

Details:

val = {
