ipfilter_policy_clone_set - PyFOS util to clone ipfilter policy¶

The ipfilter_policy_clone_set supports ‘ipfilter’ CLI use case.

This module is a standalone script and API that can be used to clone from the existing ipfilter policy configuration.

  • inputs:
Infrastructure options:
-i,–ipaddr=IPADDR IP address of FOS switch.
-L,–login=LOGIN login name.
-P,–password=PASSWORD password.
-f,–vfid=VFID VFID to which the request is directed to [OPTIONAL].
-s,–secured=MODE HTTPS mode “self” or “CA” [OPTIONAL].
-v,–verbose verbose mode[OPTIONAL].
  • Util scripts options:
    --name

    set ipfilter policy name

    --destination-name
     

    set destination name for clone
  • outputs:
    • success response or dictionary in case of error.
