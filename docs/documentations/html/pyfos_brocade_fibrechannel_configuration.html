pyfos_brocade_fibrechannel_configuration - PyFOS module to configure FC switches.¶

The pyfos_brocade_fibrechannel_configuration module provides REST support to configure FC switches.

class pyfos_brocade_fibrechannel_configuration.f_port_login_settings(dictvalues={})¶
Class of configurable parameters of the FC switch corresponding to

F_Port login.

Class Members:

Attribute Name Description Frequently Used Methods
max-logins Sets the maximum number of logins. set_max_logins() peek_max_logins()
max-flogi-rate-per-switch Sets the FLOGI rate in the switch. set_max_flogi_rate_per_switch() peek_max_flogi_rate_per_switch()
stage-interval Sets the stage interval. set_stage_interval() peek_stage_interval()
free-fdisc Sets the allowed FDISC. set_free_fdisc() peek_free_fdisc()
enforce-login Sets the login type precedence. set_enforce_login() peek_enforce_login()

Object Methods

get(session)¶

Returns a switch object with attributes from the switch.

Each object can be printed using pyfos_util.response_print(), and individual attributes can be accessed through peek methods.

Parameters:session – The session handler returned by pyfos_auth.login().
Return type:A dictionary of error or a switch object.
patch(session)¶

Applies configurable attribute(s) within the object to the switch.

Example Using Individual Sets:

# initialize the switch object
obj =
pyfos_brocade_fibrechannel_configuration.f_port_login_settings()
# set the enforce-login attribute to enable the
# login type precedence on the switch
obj.set_enforce_login(1)
# execute HTTP patch command to apply the object to the
# switch connected through session
obj.patch(session)

Example of Combining Object Initialization and Attribute Sets:

# initialize the switch object and
# set the max-logins attribute
obj =
pyfos_brocade_fibrechannel_configuration.f_port_login_settings(
    {"max-logins" : 1})
# execute HTTP patch command to apply the object to the
# switch connected through session
obj.patch(session)
Parameters:session – The session handler returned by pyfos_auth.login().
Return type:A dictionary of error or success information.
