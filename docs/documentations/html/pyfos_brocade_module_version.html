pyfos_brocade_module_version - PyFOS module to provide module level version information for all the supported modules.¶

The pyfos_brocade_module_version provides the REST support for module version.

class pyfos_brocade_module_version.module_version(dictvalues={})¶

Class of module version

Important class members:

Attribute name Description Frequently used methods
name It refers to yang module name peek_name()
version Yang module version in the format of x.y.z. peek_version()
uri Uniform Resource Identifier (URI) of a module. peek_uri()
objects List of all the supported objects from this module. peek_objects()

Object methods

classmethod get(session)¶

Returns a module_version object filled with attributes gathered throuth the session passed in. Each object can be printed using pyfos_util.response_print().

Parameters:session – session handler returned by utils.brcd_util.getsession()
Return type:a module-version object

Attribute methods

peek_name()¶

Represents the module name.

Return type:A dictionary in case of error or a success response.
