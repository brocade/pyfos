pyfos_brocade_operation_firmwaredownload - PyFOS module for the firmwaredownload operation.¶

The pyfos_brocade_operation_firmwaredownload module provides REST support for
the firmwaredownload operation.
class pyfos_brocade_operation_firmwaredownload.firmwaredownload(dictvalues={})¶

Class of the firmwaredownload Operation

Object Methods

post()¶

Configures firmwaredownload. The required fields are set within the object using the attribute’s set method. This method is used to trigger the firmwaredownload operation in the switch.

Example Usage of the Method to Post the Firmwaredownload Operation:

ss_obj =
    pyfos_brocade_operation_firmwaredownload.firmwaredownload()
ss_obj.set_host("1.1.1.1")
ss_obj.set_user_name("user")
ss_obj.set_password("abcd")
ss_obj.set_remote_directory("/a/b/c/d")
ss_obj.set_protcolo("scp")
ss_obj.set_stage(False)
ss_obj.set_activate(True)
ss_obj.set_eula_action("accept_eula")
ss_obj.post(session)

The above example triggers the firmwerdownload operation.

Parameters:session – The session handler returned by pyfos_auth.login().
Return type:A dictionary of errors or a success response.

set_host()¶

Sets the host in the object.

Parameters:host – The host name of the remote server.
Return type:A dictionary of errors or a success response.
set_user_name()¶

Sets the user name of the remote host.

Parameters:username – The user name of the remote server.
Return type:A dictionary of errors or a success response.
set_password()¶

Sets the remote username’s password.

Parameters:password – The password of the remote user.
Return type:A dictionary of errors or a success response.
set_remote_directory()¶

Sets the remote server path.

Parameters:path – The path to store the supportsave logs.
Return type:A dictionary of errors or a success response.
set_protocol()¶

Sets the protocol of the remote server access.

Parameters:protocol – The protocol to communicate with the server.
Return type:A dictionary of errors or a success response.
set_port()¶

Sets user defined port number for scp or sftp

Parameters:port – port number.
Return type:A dictionary of errors or a success response.
set_stage()¶

Sets option to download firmware only on mount partitions. With pre 9.0.0 FOS, only False is supported.

Parameters:stage – boolean.
Return type:A dictionary of errors or a success response.
set_activate()¶

Sets option to activate firmware if not already activated. With pre 9.0.0 FOS, only True is supported.

Parameters:stage – boolean.
Return type:A dictionary of errors or a success response.
set_eula_action()¶

Indicates action for BSN EULA

Parameters:choice – accept-eula, decline-eula or display-eula.
Return type:A dictionary of errors or a success response.
