pyfos_brocade_lldp - PyFOS module Proprietary Link Level DiscoveryProtocol (LLDP) entry container. This container contains LLDP configuration,operational state and statistics.¶

The pyfos_brocade_lldp The PyFOS module support Proprietary Link LevelDiscovery Protocol (LLDP) entry container. This container contains LLDPconfiguration, operational state and statistics.

class pyfos_brocade_lldp.lldp_global(dictvalues=None)¶

Class of lldp_global

Description lldp_global

The LLDP switch level configuration and operational parameters.

Important class members of lldp_global:

Attribute Name Description Frequently Used Methods
tx-interval The LLDP Tx interval of the switch. Default value is 30 seconds set_tx_interval() peek_tx_interval()
system-description The LLDP system description of the switch. set_system_description() peek_system_description()
multiplier The LLDP timeout multiplier value. Default value is 4 set_multiplier() peek_multiplier()
system-name The LLDP system name of the switch. set_system_name() peek_system_name()
tlv The list of optional TLVs enabled on the switch. The dcbx, fcoe-app, fcoe-lls, sys-name and port-desc TLVs are enabled by default and user can disable them if required. The dcbx TLV should be enabled beforehand to enable fcoe-app and fcoe-lls TLVs. set_optional_tlvs_tlv() peek_optional_tlvs_tlv()
enabled-state The LLDP protocol state of the switch. The possible values are: true : LLDP is enabled on the switch false : LLDP is disabled on the switch Default value is true. set_enabled_state() peek_enabled_state()
tlv The list of mandatory LLDP TLVs (chassis-id, port-id and time-to-live) which cannot be modified. set_mandatory_tlvs_tlv() peek_mandatory_tlvs_tlv()

Description lldp_profile

The LLDP profile.

Important class members of lldp_profile:

Attribute Name Description Frequently Used Methods
tx-interval The Transmit interval value of the LLDP profile. Tx-interval value -1 indicates that it will be derived from lldp global tx-interval value. set_tx_interval() peek_tx_interval()

multiplier The LLDP timeout multiplier value of the LLDP profile. Multiplier value -1 indicates that it will be derived from lldp global multiplier value. set_multiplier() peek_multiplier()
tlv List of enabled TLVs for the LLDP profile. Empty enabled-tlvs indicates that the tlvs will be derived from lldp global optional-tlvs. set_enabled_tlvs_tlv() peek_enabled_tlvs_tlv()
