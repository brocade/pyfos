class pyfos_brocade_extension.dp_hcl_status(dictvalues=None)¶

Class of dp_hcl_status

Description dp_hcl_status

Represents the HCL status on extension datapath process.

Important class members of dp_hcl_status:

Attribute Name Description Frequently Used Methods
dp-id Extension Data Path Processor ID. Based on platform either it will have a single DP or dual DP. In case of single DP only DP0 is supported, and in case of dual DP both DP0 and DP1 are supported 0 : DP0 1 : DP1. peek_dp_id() set_dp_id()
slot The slot number of for the datapath processor. peek_slot() set_slot()
ip-hcl-stage The current DP HCL stage for IP protocol. peek_ip_hcl_stage()
firmware-version A human readable string identifying the firmware version running on the datapath process of the switch/blade. peek_firmware_version()
state The current DP HCL state peek_state()
svi-swapped Is the SVI swapped for HCL processing peek_svi_swapped()
dp-communication-status The current state of DP-DP communication peek_dp_communication_status()
status The current DP status. peek_status()
fc-hcl-stage The current DP HCL stage for FC protocol. peek_fc_hcl_stage()
