pyfos_brocade_fibrechannel_switch - PyFOS module to provide REST support for a FC switch.¶

The pyfos_brocade_fibrechannel_switch provides a REST support for a FC switch.

get(session, name=None)¶

Returns a fibrechannel_switch object or a list of objects with attributes gathered from switch. If an optional name is given, returns either an object matching the WWN of the switch or an empty object.

Each object can be printed using pyfos_util.response_print() and individual attributes can be accessed through peek methods.

Parameters:session – The session handler returned by pyfos_auth.login().
Return type:A fibrechannel_switch object if there is only one switch within fabric or a list of objects if there is more than one switch.
patch(session)¶

Apply configurable attribute(s) within the object to the switch.

Below is an Example Using Individual Sets:

# initialize the switch object
switch = pyfos_switch.fibrechannel_switch()
# set the enabled-state attribute to enable the switch
switch.set_enabled_state(pyfos_switch.ENABLE)
# execute HTTP patch command to apply the object to the
# switch connected through session
switch.patch(session)

Below is an Example of Combining Object Initialization and Attribute Sets:

# initialize the switch object and
# set the enable-state attribute
switch = pyfos_switch.fibrechannel_switch(
    {"enabled-state" : pyfos_switch.ENABLE})
# execute HTTP patch command to apply the object to the
# switch connected through session
switch.patch(session)
Parameters:session – The session handler returned by pyfos_auth.login().
Return type:A dictionary of errors or success information.
