Object methods

get(session, name=None)¶

Returns a fibrechannel_switch object or a list of objects with attributes gathered from the switch. If an optional name is given, returns either an object matching the WWN of the switch or an empty object.

Each object can be printed using pyfos_util.response_print(), and individual attributes can be accessed through peek methods.

Parameters:session – The session handler returned by pyfos_auth.login().
Return type:A fibrechannel_switch object if there is only one switch within the fabric or a list of objects if there is more than one switch.
patch(session)¶

Applies configurable attribute(s) within the object to the switch.

Below Is an Example Using Individual Sets:

# initialize the switch object
switch = pyfos_switch.fibrechannel_switch()
# set the enabled-state attribute to enable the switch
switch.set_enabled_state(pyfos_switch.ENABLE)
# execute HTTP patch command to apply the object to the
# switch connected through session
switch.patch(session)

Below Is an Example of Combining Object Initialization and Attribute Sets:

# initialize the switch object and
# set the enable-state attribute
switch = pyfos_switch.fibrechannel_switch(
    {"enabled-state" : pyfos_switch.ENABLE})
# execute HTTP patch command to apply the object to the
# switch connected through session
switch.patch(session)
Parameters:session – The session handler returned by pyfos_auth.login().
Return type:A dictionary of errors or success information.

class pyfos_brocade_fibrechannel_switch.topology_domain(dictvalues={})¶

Class representing each domain in the fabric that is reachable from the queried switch, and the results of the most recent Fabric Shortest Path First calculation detailing how the queried switch reaches the specified destination domain.

Important Class Members:

Attribute Name Description Frequently Used Methods
domain-id Destination fabric domain. peek_domain_id()
path-count The number of lowest, equal cost paths available. peek_path_count()
metric Lowest path cost of the in-use paths to reach the domain. peek_metric()
is-local-translate-domain If the destination domain is reachable via a local EX-Port. peek_is_local_translate_domain()
out-ports The local ports used for routing traffic to the domain. peek_out_ports()

Object methods

get(session, domain_id=None)¶

Returns a topology_domain object with details from the switch, list of topology_domain objects, or a dictionary with error information. If the optional domain_id parameter is given, the corresponding topology_domain object is returned, or a dict of error information.

Each object can be printed using pyfos_util.response_print(), and individual attributes can be accessed through peek methods.

Parameters:
  • session – The session handler returned by pyfos_auth.login().
  • domain_id (int, optional) – The destination domain for which to get local out ports to use for reaching the domain. Defaults to None.
Return type:

A topology_domain object, list of topology_domain objects, or dictionary with error information.

Attribute Methods

peek_domain_id()¶

Gets the domain ID in the object.

Return type:None or an integer of the domain ID
peek_path_count()¶

Gets the number of lowest, equal cost paths available as of the last Fabric Shortest Path First calculations.

Return type:None or an integer of the number of paths
peek_metric()¶

Gets the path cost associated with routing to domain.

Return type:None or an integer of the path cost.
peek_is_local_translate_domain()¶

Gets a flag indicating if the domain is reachable via a local EX-Port.

Return type:None or boolean
peek_out_ports()¶

Gets the list of local ports that are programmed for routing traffic to the domain.

Return type:A dictionary with a list of the out ports as strings
class pyfos_brocade_fibrechannel_switch.topology_route(dictvalues={})¶

Class containing a mapping of how the switch’s ports are routed, as ingress ports, to their corresponding egress ports for frames destined to the specified destination domain. Reports the greatest number of hops across all the equal cost paths to reach the specified destination domain.

Important Class Members:

Attribute Name Description Frequently Used Methods
domain-id Destination fabric domain for the shown routes. peek_domain_id()
hop-count Greatest number of hops among all equal cost paths used by this switch to reach the domain. peek_hop_count()
out-port The egress port for these ingress ports, for traffic to the domain. peek_out_port()
in-ports The switch’s ports that are routed this out-port for frames destined to the domain. peek_in_ports()

Object methods

get(session)¶

Returns a list of topology_route objects, an empty topology_route object, or a dictionary with error information.

Each object can be printed using pyfos_util.response_print(), and individual attributes can be accessed through peek methods.

Parameters:session – The session handler returned by pyfos_auth.login().
Return type:A list of topology_route objects, an empty topology_route object, or dictionary with error information.
