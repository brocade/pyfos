Object methods

static get(session, port_id=None)¶

Returns a fibrechannel_name_server object or a list of objects filled with Name Server attributes.

Each object can be printed using pyfos_util.response_print() and individual attributes accessed through peek methods.

Parameters:session – session handler returned by pyfos_auth.login()
Return type:port-id object or list of objects. Dictionary in case of error.
