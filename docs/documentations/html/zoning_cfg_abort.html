zoning_cfg_abort - PyFOS util for specific Zoning use case.¶

The zoning_cfg_abort provides for a specific Zoning use case.

This module is a stand-alone script and API that can be used to abort pending changes to a Zone DB.

  • Inputs:
    • -L=<login>: Login ID. If not provided, an interactive

      prompt will request one.

    • -P=<password>: Password. If not provided, an interactive

      prompt will request one.

    • -i=<IP address>: IP address.

    • -f=<VFID>: VFID or -1 if VF is disabled. If unspecified,

      a VFID of 128 is assumed.

  • Outputs:
    • Python dictionary content with RESTCONF response data.
zoning_cfg_abort.cfgabort(session)¶

Abort pending Zone DB transaction.

Example usage of the method:

result = cfgabort(session)
Parameters:session – session returned by login.
Return type:dictionary of return status matching rest response.

Use cases

Abort pending Zone DB changes.
