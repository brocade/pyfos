fdmi_show - PyFOS util to show FMDI entries.¶

The fdmi_show supports ‘fdmishow’ CLI use case.

This module is a standalone script and API that can be used to display all FDMI HBA and port entries. If an HBA ID is specified, only the entries corresponding to that HBA will be displayed.

  • Inputs:
    • -L=<login>: Login ID. If not provided, an interactive

      prompt will request one.

    • -P=<password>: Password. If not provided, an interactive

      prompt will request one.

    • -i=<IP address>: IP address.

    • -f=<VFID>: VFID or -1 if VF is disabled. If unspecified,

      a VFID of 128 is assumed.

    • –hbaid=<HBAID>: HBA ID.

  • Outputs:
    • Python dictionary of FDMI HBA entries and Python dictionary of

      FDMI HBA port entries.
