log_setting_keep_alive_period_show - PyFOS util for setting the keepalive period.¶

The log_setting_keep_alive_period_show util provides for configuring the keepalive period.

This module is a stand-alone script that can be used to display log setting attributes.

  • Input:
    • -L=<login>: The login ID. If not provided, an interactive
      prompt will request one.
    • -P=<password>: The password. If not provided, an interactive
      prompt will request one.
    • -i=<IP address>: The IP address.
    • -f=<VFID>: The VFID or -1 if VF is disabled. If unspecified,
      a VFID of 128 is assumed.
  • Output:
    • Log setting attributes in JSON format.
log_setting_keep_alive_period_show.show_alive_period(session)¶

Example Usage of the Method:

ret = log_setting_keep_alive_period_show.show_alive_period(session)
print (ret)

Details:

result = log_setting_keep_alive_period_show.show_alive_period(
        session)
  • Input:
    param session:The session returned by the login.
  • Output:
    rtype:A dictionary of return status matching the REST response.

Use Cases

  1. Retrieve the keepalive period of the RASLog.
