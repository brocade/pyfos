log_setting_keep_alive_period_set - PyFOS util for setting the keepalive period.¶

The log_setting_keep_alive_period_set util provides for setting the keepalive period.

This module is a stand-alone script that can be used to set the keepalive period.

  • Input:
    • -L=<login>: The login ID. If not provided, an interactive

      prompt will request one.

    • -P=<password>: The password. If not provided, an interactive

      prompt will request one.

    • -i=<IP address>: The IP address.

    • -f=<VFID>: The VFID or -1 if VF is disabled. If unspecified,

      a VFID of 128 is assumed.

  • Util Script Options:
    • --alive=ALIVE-PERIOD
       

      Sets the alive period for the login hours.
  • Output:
    • Log setting attributes in JSON format.

ret = log_setting_keep_alive_period_set.set_alive_period(session,
