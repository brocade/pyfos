Object methods

static get(session, hba_identifier=None)¶

Returns a hba object or a list of objects filled with FDMI HBA attributes.

Each object can be printed using pyfos_util.response_print() and individual attributes accessed through peek methods.

Parameters:session – session handler returned by pyfos_auth.login()
Return type:hba object or list of objects. Dictionary in case of error.
