extensionShell - PyFOS util for emulating an extension shell.Â¶

The extensionShell util provides CLI kinds of shell functionality.

This module is a stand-alone script that can be used for basic operation using an extension CLI. It does not cover all the supported CLI commands.

  • Supported Objects:
    • ipif
    • iproute
    • fciptunnel
    • fcipcircuit
  • Operations:
    • portcfg
    • portshow

Example:

portshow ipif
portshow iproute
portshow fciptunnel
portshow fcipcircuit
