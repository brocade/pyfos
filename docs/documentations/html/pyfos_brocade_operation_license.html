pyfos_brocade_operation_license - PyFOS module for the license operation.¶

The pyfos_brocade_operation_license module provides REST support for
the license operation.
class pyfos_brocade_operation_license.license(dictvalues={})¶

Class of the license Operation

Important Class Members:

Attribute Name Description Frequently Used Methods
name license name. set_name()
action install or remove license. set_action()

Object Methods

post()¶

Configures license. The required fields are set within the object using the attribute’s set method.

Example Usage of the Method to Post the license Operation:

license_obj =
    pyfos_brocade_operation_license.license()
license_obj.set_action("install")
license_obj.set_name("XYZLICENSEKEY")
license_obj.post(session)

The above example triggers the license install operation.

Parameters:session – The session handler returned by pyfos_auth.login().
Return type:A dictionary of errors or a success response.
