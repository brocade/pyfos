zoning_alias_remove - PyFOS util for alias remove use case¶

The zoning_alias_remove supports an alias remove use case.

This module is a stand-alone script and API that can be used to remove members from an existing alias(es).

  • Inputs:
    • -L=<login>: Login ID. If not provided, an interactive

      prompt will request one.

    • -P=<password>: Password. If not provided, an interactive

      prompt will request one.

    • -i=<IP address>: IP address.

    • –name=<alias name>: string name of an existing alias

    • –members=<member list>: list of members separated by “;”.

      Multiple members need to be enclosed by “”.

    • -f=<VFID>: VFID or -1 if VF is disabled. If unspecified,

      a VFID of 128 is assumed.

  • Outputs:
    • Python dictionary content with RESTCONF response data.
zoning_alias_remove.aliasremove(session, aliases)¶

Remove members from existing alias(es).

Example usage of the method:

aliases = [
            {
                "alias-name": name,
                "member-entry": {"alias-entry-name": members}
            }
          ]
result = aliasremove(session, aliases)
Parameters:
  • session – session returned by login.
  • aliases – an array of alias and to be removed members.
Return type:

Dictionary of return status matching rest response.

Use cases

Remove members from an existing alias.
