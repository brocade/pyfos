pyfos_brocade_extension_ipsec_policy - PyFOS module for an IPsec policy.¶

The pyfos_brocade_extension_ipsec_policy module provides REST support for an extension IPsec policy.

class pyfos_brocade_extension_ipsec_policy.extension_ipsec_policy(dictvalues={})¶

profile-name The profile name of the policy. peek_profile_name() set_profile_name()
